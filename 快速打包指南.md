# 快速打包mobile_payment_simple.html为安卓应用

## 一、前提条件确认

### 必需工具
- [ ] Node.js已安装
- [ ] npm已安装
- [ ] Cordova已安装
- [ ] Java JDK已安装
- [ ] Android SDK已安装

### 快速检查命令
```bash
node -v
npm -v
cordova -v
java -version
adb version
```

## 二、快速创建Cordova项目

### 步骤1：创建项目
```bash
# 在命令提示符中执行
cordova create PaymentApp com.example.payment PaymentApp
```

### 步骤2：进入项目
```bash
cd PaymentApp
```

### 步骤3：添加Android平台
```bash
cordova platform add android
```

### 步骤4：检查环境
```bash
cordova requirements android
```

## 三、配置Web应用

### 步骤1：进入www目录
```bash
cd www
```

### 步骤2：创建index.html
创建`index.html`文件，内容如下：

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
    <meta name="format-detection" content="telephone=no">
    <title>支付页面</title>
    <style>
        body {
            margin: 0;
            padding: 0;
            font-family: Arial, sans-serif;
            background-color: #f5f5f5;
        }
        .loading {
            display: flex;
            justify-content: center;
            align-items: center;
            height: 100vh;
            flex-direction: column;
        }
        .spinner {
            width: 50px;
            height: 50px;
            border: 5px solid #f3f3f3;
            border-top: 5px solid #007bff;
            border-radius: 50%;
            animation: spin 1s linear infinite;
        }
        @keyframes spin {
            0% { transform: rotate(0deg); }
            100% { transform: rotate(360deg); }
        }
    </style>
    <script>
        window.onload = function() {
            setTimeout(function() {
                window.location.href = "http://192.168.100.174:5000/mobile_payment_simple";
            }, 1000);
        };
    </script>
</head>
<body>
    <div class="loading">
        <div class="spinner"></div>
        <p style="margin-top: 20px; color: #666;">正在加载支付页面...</p>
    </div>
</body>
</html>
```

### 步骤3：安装插件
```bash
cordova plugin add cordova-plugin-network-information
cordova plugin add cordova-plugin-device
```

## 四、构建APK

### 步骤1：构建调试版
```bash
cordova build android
```

### 步骤2：构建发布版
```bash
cordova build android --release
```

### 步骤3：查找APK文件
调试版：`PaymentApp\platforms\android\app\build\outputs\apk\debug\app-debug.apk`
发布版：`PaymentApp\platforms\android\app\build\outputs\apk\release\app-release-unsigned.apk`

## 五、签名APK（发布需要）

### 步骤1：生成密钥库
```bash
keytool -genkey -v -keystore my-release-key.keystore -alias my-key-alias -keyalg RSA -keysize 2048 -validity 10000
```

### 步骤2：创建build.json
在项目根目录创建`build.json`文件：

```json
{
    "android": {
        "release": {
            "keystore": "my-release-key.keystore",
            "alias": "my-key-alias",
            "storePassword": "你的密钥库密码",
            "password": "你的别名密码"
        }
    }
}
```

### 步骤3：重新构建
```bash
cordova build android --release
```

## 六、测试APK

### 步骤1：传输到设备
通过USB连接或传输APK文件到安卓设备

### 步骤2：安装APK
在设备上打开APK文件并安装

### 步骤3：测试功能
- 检查是否能正常加载支付页面
- 检查支付流程是否正常
- 检查页面显示是否正常

## 七、常见问题

### 问题1：cordova命令未找到
- 解决：检查Cordova是否已全局安装

### 问题2：Android target not installed
- 解决：使用Android Studio安装Android SDK Platform

### 问题3：Gradle not installed
- 解决：配置GRADLE_HOME环境变量

### 问题4：页面无法访问
- 解决：检查网络连接，确保设备与服务器在同一网络
